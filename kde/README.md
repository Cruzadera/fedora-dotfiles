# KDE exports

This directory is a landing zone for KDE Plasma backups.

## Recommended exports

- `plasma/`: Plasma panel and widget layout
- `kwin/`: KWin scripts, rules, and shortcuts
- `konsole/`: Konsole profiles and color schemes
- `color-schemes/`: KDE color scheme files
- `icons/`: custom icon themes or overrides
- `wallpapers/`: desktop wallpapers

## Notes

- Exported files vary slightly across Fedora and KDE releases.
- Keep this folder free of secrets and browser profile data.
