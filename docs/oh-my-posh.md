# Oh My Posh

This repository ships the `roma-gruvbox` theme.

## Restore

Copy or link `oh-my-posh/roma-gruvbox.omp.json` to your config directory and initialize it from Bash.

## Theme notes

- Uses a Gruvbox-style palette.
- Keeps the main prompt compact.
- Highlights Git state and active language tooling without overwhelming the terminal.
