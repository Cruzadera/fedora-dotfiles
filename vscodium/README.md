# VSCodium

This folder stores the portable part of the VSCodium configuration.

## Included files

- `settings.json`
- `keybindings.json`
- `extensions.txt`
- `snippets/`
- `profiles/`

## Notes

- Do not commit authentication tokens, signed-in sessions, or account caches.
- Workspace settings can be added in a separate, project-specific folder if needed.
